TABVOID - Get Really Real!

Tabvoid is some javascript code written for weblogging. It aims to be a flexible solution for my fellow impulsive (re)designers.

Tabvoid works similarly with Zonelets, with a few minor difference such as:

  • The body tags only contain the post's content while the pages format is inputted using javascript.
  • Posts data is stored in an object instead of an array; this allows more customization.
  • It assumes that you are already familiar with html and css.
